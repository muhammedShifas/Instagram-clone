# Instagram-clone
A responsive desktop clone of Instagram built using HTML, CSS, and JavaScript. This project recreates the core Instagram desktop interface for educational and portfolio purposes.
# Instagram Desktop Clone

## Features

* Responsive desktop layout
* Instagram-inspired user interface
* Navigation sidebar
* Stories section
* Feed posts
* Profile suggestions
* Clean and modern design
* Interactive hover effects and animations

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
Instagram-Clone/
│
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/Instagram-Clone.git
```

2. Open the project folder:

```bash
cd Instagram-Clone
```

3. Open `index.html` in your browser.
